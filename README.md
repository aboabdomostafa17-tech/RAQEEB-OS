RAQEEB-OS is an advanced AI-powered security and attendance system. It utilizes Face Recognition technology to monitor premises, log attendance automatically, and provide real-time security alerts.

## 🚀 Key Features
* Face Recognition: Identifies registered individuals in real-time.
* Automated Logging: Saves entry/exit data directly into an Excel sheet.
* Telegram Integration: Sends instant photo alerts to your phone when an intruder is detected.
* Sound Alarm: Triggers an audible warning for unauthorized access.

## 🛠️ Built With
* Python
* OpenCV & Face_Recognition API
* Telepot (Telegram Bot)
* Pandas (Excel Management)
